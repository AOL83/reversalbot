# 07 - Operating Manual

## Startup Checklist
1. Validate configuration.
2. Confirm kill switch functionality.
3. Verify broker connectivity.

## Manual Controls
- **pause:** stop routing new orders.
- **resume:** allow routing if risk checks pass.
- **flatten:** cancel outstanding orders and neutralize exposure.
- **kill:** engage the kill switch immediately.
- **reset-kill:** only after verifying safety.
